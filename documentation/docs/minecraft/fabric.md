# mc.andrewlee.fun

This server is designed to be a Vanilla+ experience with an emphasis on having a good time, not a grind time.


Many of the mods are designed to take the hurt out of playing Minecraft. *`Villager-Pickup`* makes moving villagers zero hassle. *`Vein-Miner-Enchantment`* makes mining veins a breeze. *`Tree-Harvester`* makes deforestation quick and easy. *`TNT-Breaks-Bedrock`* is a must-have for getting on top of the Nether. And so much more!


Whitelist is invite only.

## Join the Server

`mc.andrewlee.fun` is a Minecraft server running on the machine [`kamrui-p1`](https://github.com/andrewthomaslee/home/blob/main/machines/kamrui-p1/configuration.nix) and accessible via the [playit.gg](https://playit.gg) proxy with the raw url: `wide-sell.gl.joinmc.link`.


Some mods are required to join the server.


### Requirements

- Minecraft Java Version `1.21.11`
- Fabric Loader Version `0.18.4`
