# Firefox Module

Welcome to the **Firefox** module documentation! Bring the privacy-respecting, lightning-fast web browser to your daily workflow.

## Overview

This lightweight module handles the declarative installation of Mozilla Firefox via Home Manager, ensuring the browser is always available and up-to-date in your user environment.

## Usage

Enable the module in your Home Manager configuration:

```nix
homeSpec.programs.firefox.enable = true;
```
