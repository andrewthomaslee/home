# Ghostty Module

Welcome to the **Ghostty** module documentation! Experience the future of terminal emulators with blistering speed and gorgeous aesthetics.

## Overview

This module installs and configures Ghostty, a modern, GPU-accelerated terminal emulator, straight from the unstable channel to ensure you have the absolute latest features.

## Key Features

- **Cyberpunk Aesthetics:** Pre-configured with the stunning "Cyberpunk Scarlet Protocol" theme, providing a visually striking and high-contrast environment for long coding sessions.
- **Perfect Typography:** Hardcodes a comfortable, highly readable font size of 15, preventing eye strain.

## Usage

Enable the module in your Home Manager configuration:

```nix
homeSpec.programs.ghostty.enable = true;
```
