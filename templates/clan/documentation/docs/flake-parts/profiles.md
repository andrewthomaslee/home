# flake-parts/profiles.nix

Host and user profiles.

## Overview
Organizes configurations into reusable profiles for different machine types and users.

## NixOS Profiles
- `profile-developer`: Andrew's PC configuration.
- `profile-server`: Headless server configuration.
- `profile-normal`: Standard PC configuration.

## Home Manager Profiles
- `profile-developer`: Developer toolset.
- `profile-server`: Server utilities.
- `profile-normal`: Basic desktop applications.
